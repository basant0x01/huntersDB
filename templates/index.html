HackerOne Programs
ℹ About HackerOne Sync

The HackerOne sync will:

  1. Fetch all open programs you have access to via the H1 API
  2. Run subfinder on each domain in scope
  3. Run HTTPX alive check (Phase A)
  4. Update subdomain lifecycle states (Phase B)
  5. Deep HTTPX enrichment (Phase C)
  6. Queue Nuclei for manual execution per program
Nuclei runs manually only for private programs. Use the Nuclei Queue above.

YesWeHack Programs
ℹ About YesWeHack Sync

The YesWeHack sync will:

  1. Paginate all public + private programs via YWH API
  2. Fetch full scope for each program (web-app + API targets)
  3. Run subfinder for subdomain discovery
  4. Run HTTPX alive check → lifecycle update → deep scan
  5. Queue Nuclei for manual execution per program
Nuclei runs manually only for private programs — use the Nuclei Queue above or the Nuclei Scanner page.
